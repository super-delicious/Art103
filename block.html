Block


Part 1

The following models are an exercise after watching the fusion 360 tutorials is to learn the concepts and methods for making sold model objects. I've never used this program before, so the tutorials videos are helpful. The first model is based on Lesson 5, and I created this model to sketch and revolve. The second model is based on Lesson 7; I call this " avocado knife," I made this model started with a box the used fillet and extruded the side, then cut the hole and repeated the shape. After modeling, I give them different physical materials and appearances that stimulate the physical object.

Part 2

The second part of the project is to create two separate objects. The first object is freestyle and contains a range of features. I designed a table and decorations. The second object is constrained, and I choose to make a model of an infrared thermometer. I select the constraints are triangular sequences. I included each screenshot with each process's timeline bar to show my workflow.
